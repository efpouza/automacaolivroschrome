Raspador de Livros - Loja Virtual
Este projeto é uma aplicação web desenvolvida com Streamlit que permite a raspagem de dados de livros disponíveis no site Books to Scrape. A aplicação coleta informações como título, preço, disponibilidade e avaliação dos livros e permite o download dos dados em formato CSV.

Funcionalidades
Interface web interativa com Streamlit
Raspagem de dados automatizada com requests e BeautifulSoup
Barra de progresso dinâmica durante a coleta
Exibição em tempo real da porcentagem e status de coleta
Visualização dos dados em uma tabela
Opção para baixar os dados em arquivo .csv
Botão para reiniciar/limpar a coleta

Tecnologias Utilizadas
Python
Streamlit
Pandas
Requests
BeautifulSoup
